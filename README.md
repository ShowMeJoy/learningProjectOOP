# Практическое задание по дисциплине "Алгоритмизация и программирование"

Для заданной предметной области разработать:
1) UML-диаграмму классов;
2) программный код заданных классов на С++;
3) программный код методов модификации объектов с контролем всех входных параметров;
4) программный код метода соответственно таблице индивидуальных заданий;
5) программу тестирования разработанных классов.

Задание:
- Защита информации;
- Сетевой адрес компьютера, лицензия на программное обеспечение, пользователь;
- Найти сетевые адреса компьютеров для указанных пользователей, на которых установлено ПО с заданными номерами лицензий
